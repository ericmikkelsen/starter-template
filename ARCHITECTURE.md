# Template Architecture

This repository is a lightweight GitHub template focused on conventions, automation,
and reusable AI workflow guidance.

## Scope

- Coding standards and project conventions
- GitHub workflows for commit hygiene and release automation
- Reusable skill references in `.github/skills/`
- Copilot chat mode instructions in `.github/chat-modes/`

## Design Goal

Keep template repos lean and tool-agnostic. Projects created from this template should
inherit conventions and workflows without inheriting editor-specific runtime dependencies.

## Skills Docs

The `.github/skills/` SKILL.md files are canonical workflow references. They describe
what each workflow does, when to use it, and verification criteria.
